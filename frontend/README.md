# SRM Credit Engine — Frontend

Painel do Operador e Grid de Transações da plataforma de cessão de crédito. React + TypeScript, build com Vite.

Ver [`README.md`](../README.md) na raiz do repositório para visão geral do projeto, stack completa e status atual.

## Como rodar

```bash
npm install
npm run dev      # http://localhost:5173
```

## Scripts

| Comando | O que faz |
|---|---|
| `npm run dev` | Sobe o servidor de desenvolvimento (HMR) |
| `npm run build` | Type-check (`tsc -b`) + build de produção |
| `npm run lint` | Lint (oxlint) |
| `npm test` | Testes (Vitest + Testing Library) |
| `npm run preview` | Serve o build de produção localmente |
